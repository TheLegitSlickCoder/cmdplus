CMD+ Documentation:

Getting Started:
1. Find the directory cmd+.exe is located in.
2. Change the directory in your command line to that directory.
3. You can now execute cmd+ commands using cmd+.exe.

Example:
cmd+ read C:\Users\James\Documents\file.txt

Commands:
read (path): Displays all the contents of a file on the console.
contents (path): Displays all the contents of a directory.
createfile (path): Creates a file.
launch (path/URL): Launches a path or URL.
checkpath (path): Checks if a path exists.
length (path): Gets the length of a file.
